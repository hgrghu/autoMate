# AI增强富文本编辑器使用指南

## 📖 项目概述

**AI增强富文本编辑器** 是基于 autoMate 项目构建的智能文本编辑器，集成了强大的 LLM 辅助编辑功能。它不仅提供了传统富文本编辑器的所有功能，还通过 AI 技术极大地提升了文本创作和编辑的效率。

### 🌟 核心特性

- 🎨 **完整的富文本编辑功能**
- 🤖 **AI 辅助文本处理**（润色、扩写、翻译、总结等）
- 🖼️ **智能图片处理**
- 📊 **图表编辑支持**
- 🗂️ **文档大纲管理**
- 📝 **版本历史控制**
- 📤 **多格式导出**（Word、PDF、Markdown、HTML、纯文本）

---

## 🚀 安装与启动

### 环境要求
- Python 3.8+
- PyQt6
- 其他依赖包（见 `editor_requirements.txt`）

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/yuruotong1/autoMate.git
cd autoMate
```

2. **安装编辑器依赖**
```bash
pip install -r editor_requirements.txt
```

3. **启动编辑器**
```bash
python run_editor.py
```

### 配置说明

首次启动时，请在**设置**中配置：
- **API Key**: OpenAI 或其他支持的 LLM 服务密钥
- **Base URL**: API 服务地址
- **模型**: 使用的 LLM 模型（如 gpt-4o）

## 🎯 功能详解

### 1. 文本编辑功能

#### 基础编辑
- **字体设置**: 字体族、大小、颜色
- **格式化**: 加粗、斜体、下划线
- **对齐方式**: 左对齐、居中、右对齐
- **列表**: 有序列表、无序列表
- **缩进**: 增加/减少缩进

#### 快捷键
- `Ctrl+N`: 新建文档
- `Ctrl+O`: 打开文档
- `Ctrl+S`: 保存文档
- `Ctrl+Z`: 撤销
- `Ctrl+Y`: 重做
- `Ctrl+B`: 加粗
- `Ctrl+I`: 斜体
- `Ctrl+U`: 下划线

### 2. AI 辅助功能

#### 2.1 选中文本操作
选中任意文本后，可通过右键菜单或侧边栏执行：

**文本润色**
- 改善表达和语法
- 风格转换（正式、非正式、幽默等）
- 语法检查和修正

**内容扩展**
- 扩写：增加细节和深度
- 缩写：提取核心要点
- 重写：保持原意，改变表达

**翻译服务**
- 支持多种语言互译
- 英文、中文、日文、韩文等
- 保持格式和语境

**智能总结**
- 提取关键信息
- 生成简洁摘要
- 突出重点内容

**智能问答**
- 对选中内容提问
- 获得详细解释
- 理解复杂概念

#### 2.2 全文操作

**文档摘要**
- 分析全文结构
- 生成核心摘要
- 提取关键论点

**大纲生成**
- 自动提取标题层级
- 生成文档结构
- 章节导航

**内容审核**
- 敏感词检测
- 逻辑漏洞识别
- 内容合规检查

**风格统一**
- 检查语言风格一致性
- 术语使用规范化
- 语调协调性分析

### 3. 图片与媒体处理

#### 图片操作
- **插入图片**: 支持 PNG、JPG、GIF 等格式
- **尺寸调整**: 精确调整图片大小
- **位置控制**: 拖拽移动、文字环绕
- **拖拽支持**: 直接拖拽图片文件到编辑器

#### AI 图片辅助
- **描述生成**: 自动生成图片说明文字
- **智能裁剪**: AI 建议最佳裁剪区域
- **风格建议**: 根据文档风格推荐图片处理方案

### 4. 表格处理

#### 表格编辑
- **插入表格**: 自定义行列数
- **表格样式**: 边框、颜色、对齐
- **内容编辑**: 实时编辑单元格内容
- **结构调整**: 增删行列

#### 智能表格功能
- **数据分析**: AI 分析表格数据趋势
- **格式建议**: 智能推荐表格样式
- **内容生成**: 根据描述生成表格结构

### 5. 文档管理

#### 大纲视图
- **层级结构**: 自动识别标题层级
- **快速导航**: 点击跳转到指定章节
- **结构编辑**: 拖拽调整章节顺序
- **章节管理**: 添加、删除、编辑章节

#### 版本控制
- **自动保存**: 每30秒自动保存版本
- **版本历史**: 完整的修改历史记录
- **版本对比**: 查看不同版本间的差异
- **版本恢复**: 一键恢复到任意历史版本

### 6. 导出功能

#### 支持格式
- **Word (.docx)**: 保持格式完整性
- **PDF (.pdf)**: 固定排版，适合分享
- **Markdown (.md)**: 纯文本结构化格式
- **HTML (.html)**: 网页格式，在线发布
- **纯文本 (.txt)**: 提取所有文本内容

#### 导出特性
- **样式保持**: 尽可能保留原始格式
- **图片处理**: 正确嵌入图片资源
- **表格转换**: 准确转换表格结构
- **链接保持**: 保留超链接信息

## 📚 使用技巧

### 1. 高效写作流程

1. **大纲先行**: 使用 AI 生成文档大纲
2. **逐段创作**: 利用 AI 扩写功能丰富内容
3. **整体优化**: 使用风格统一功能调整语调
4. **最终润色**: 通过 AI 润色提升表达质量

### 2. AI 功能最佳实践

**选择性处理**
- 选中具体段落而非整篇文档
- 明确指定处理目标和要求
- 分步骤处理复杂任务

**迭代优化**
- 多次使用 AI 功能逐步改进
- 结合人工判断和 AI 建议
- 保存重要版本节点

### 3. 版本管理策略

**里程碑保存**
- 重要修改前手动保存版本
- 为版本添加有意义的注释
- 定期清理不必要的历史版本

**协作工作**
- 导出为通用格式共享
- 使用版本注释记录修改原因
- 建立团队版本命名规范

## ❓ 常见问题

### Q1: AI 功能无法使用怎么办？
**A**: 检查以下设置：
1. 确认已配置正确的 API Key
2. 检查网络连接是否正常
3. 验证 API 服务是否可用
4. 查看错误提示信息

### Q2: 如何提高 AI 处理质量？
**A**: 建议：
1. 提供清晰的上下文信息
2. 选择合适长度的文本片段
3. 使用具体的指令和要求
4. 多次迭代优化结果

### Q3: 导出的文档格式有问题？
**A**: 解决方案：
1. 检查是否安装了相关依赖包
2. 尝试不同的导出格式
3. 简化复杂的格式和样式
4. 查看错误日志信息

### Q4: 编辑器运行缓慢怎么办？
**A**: 优化方法：
1. 关闭不必要的功能面板
2. 减少自动保存频率
3. 清理版本历史记录
4. 重启编辑器应用
