### ⚡ 一键构建 (3种方法)

#### 方法1：Windows快速构建 (推荐)
```cmd
# 双击运行
quick_build.bat

# 或在命令行运行
./quick_build.bat
```

#### 方法2：完整自动化构建
```bash
# 安装PyInstaller
pip install pyinstaller>=5.0 pillow>=9.0.0

# 运行完整构建脚本
python build_exe.py
```

#### 方法3：手动构建
```bash
# 1. 安装依赖
pip install pyinstaller pillow
pip install -r editor_requirements.txt

# 2. 创建图标
python create_icon.py

# 3. 构建exe
pyinstaller --clean --noconfirm ai_editor.spec
```
