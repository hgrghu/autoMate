# 🚀 AI富文本编辑器 v2.0.0 - 快速开始

## 📋 系统要求

- **操作系统**: Windows 10/11 (推荐)
- **Python版本**: 3.8 或更高版本
- **内存**: 4GB+ (推荐8GB)
- **磁盘空间**: 2GB+ 可用空间

## ⚡ 一键构建 (推荐)

### Windows用户
```cmd
# 双击运行构建脚本
build_windows.bat
```

## 📦 手动构建

如果自动构建失败，可以手动执行以下步骤：

### 1. 环境准备
```cmd
# 检查Python版本
python --version

# 安装构建工具
pip install pyinstaller>=6.0 pillow>=9.0.0

# 安装项目依赖
pip install -r editor_requirements.txt
```

### 2. 创建图标
```cmd
python create_icon.py
```

### 3. 构建exe文件
```cmd
pyinstaller --clean --noconfirm ai_editor.spec
```

## 🎯 验证构建

构建完成后，可执行文件位于：
```
dist/AI富文本编辑器.exe
```

## 🔧 功能特色

### 🆕 AI增强功能
- **🤖 智能文档生成器**: 通过自然语言描述生成专业文档
- **🎨 智能排版设计**: AI分析并应用最佳排版样式

- **富文本编辑**: 完整的格式化支持

   - 重启应用程序

3. **测试功能**

## ❓ 常见问题

### 构建问题
- **Python版本过低**: 升级到3.8+
- **依赖安装失败**: 检查网络连接，尝试使用镜像源
- **杀毒软件阻止**: 添加项目目录到白名单

### 运行问题
- **启动缓慢**: 首次运行需要初始化，属正常现象
- **AI功能不可用**: 需要配置API密钥
- **文件导出失败**: 检查目标目录权限
