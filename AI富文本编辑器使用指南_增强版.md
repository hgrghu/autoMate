# AI富文本编辑器使用指南 - 增强版

## 项目概述

AI富文本编辑器是基于autoMate项目构建的智能文档处理工具，集成了先进的AI技术和完善的文档编辑功能。本增强版在原有功能基础上，新增了AI文档生成、智能排版设计和模板管理等强大功能，为用户提供全方位的智能化文档创作体验。

### 核心特性

- **智能AI辅助**: 集成GPT等大语言模型，提供文本润色、扩写、翻译等功能
- **AI文档生成**: 根据自然语言描述自动生成各种类型的专业文档
- **智能排版设计**: AI分析文档类型并自动应用最适合的排版样式
- **丰富模板库**: 提供学术论文、商业计划书、技术文档等专业模板
- **富文本编辑**: 支持字体、颜色、对齐、列表、表格等完整格式化功能
- **多格式导出**: 支持Word、PDF、Markdown、HTML等多种格式导出
- **版本管理**: 自动保存编辑历史，支持版本回溯和恢复
- **实时大纲**: 自动提取文档结构，支持快速导航

## 界面布局

## 安装指南

### 环境要求

- Python 3.8 或更高版本
- PyQt6 图形界面库
- 网络连接（用于AI功能）

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone [项目地址]
   cd automate-editor
   ```

2. **安装依赖**
   ```bash
   pip install -r editor_requirements.txt
   ```

3. **配置API密钥**
   
   首次运行时，需要配置AI服务的API密钥：
   - 在主程序中进入设置页面
   - 填入OpenAI API Key或其他AI服务的配置信息
   - 保存配置

4. **启动编辑器**
   ```bash
   # 独立启动
   python run_editor.py
   
   # 或集成到autoMate主程序
   python integration.py
   ```

## 主要功能介绍

### 1. 基础文本编辑

#### 1.1 富文本格式化

编辑器提供完整的富文本编辑功能：

- **字体设置**: 支持多种字体选择和大小调整
- **文本样式**: 加粗、斜体、下划线、删除线
- **颜色设置**: 文字颜色和背景色自定义
- **对齐方式**: 左对齐、居中、右对齐、两端对齐
- **行间距**: 可调节的行间距设置
- **缩进控制**: 增加/减少缩进

#### 1.2 列表和表格

- **无序列表**: 支持多种符号样式
- **有序列表**: 数字和字母编号
- **表格插入**: 可自定义行列数的表格
- **表格编辑**: 支持单元格合并、边框设置等

#### 1.3 图片处理

- **图片插入**: 支持拖拽和粘贴插入图片
- **尺寸调整**: 可视化的图片尺寸调整
- **位置控制**: 图片的对齐和环绕设置

### 2. AI智能功能

#### 2.1 文本处理功能

选中文本后，右键菜单提供以下AI功能：

**文本润色**
- 功能：优化文本表达，提升语言质量
- 使用：选中需要润色的文本，右键选择"润色文本"
- 效果：保持原意的基础上，使语言更加流畅和专业

**文本扩写**
- 功能：根据原文内容进行合理扩展
- 使用：选中简短的文本，选择"扩写文本"
- 效果：增加细节描述，丰富内容层次

**文本缩写**
- 功能：提炼文本核心内容，压缩篇幅
- 使用：选中较长的文本，选择"缩写文本"
- 效果：保留关键信息，简化表达

**文本翻译**
- 功能：将文本翻译为目标语言
- 使用：选中文本，选择"翻译文本"
- 支持：中英文互译，可扩展其他语言

**内容总结**
- 功能：提取文本要点，生成摘要
- 使用：选中长文本，选择"总结内容"
- 应用：快速理解文档内容，生成摘要

#### 2.2 全文处理功能

通过AI菜单或工具栏访问：

**智能摘要生成**
- 自动分析整个文档内容
- 生成结构化的摘要
- 支持不同长度的摘要需求

**章节结构生成**
- 分析文档内容，建议章节划分
- 自动生成目录结构
- 优化文档逻辑层次

**风格一致性检查**
- 检测文档中的语言风格差异
- 提供统一风格的修改建议
- 确保文档整体协调性

**内容审核**
- 检查语法错误和表达问题
- 识别可能的逻辑漏洞
- 提供改进建议

### 3. 🆕 AI文档生成器

#### 3.1 功能概述

AI文档生成器是本次更新的重点功能，支持通过自然语言描述自动生成各种类型的专业文档。

#### 3.2 使用方法

1. **打开生成器**
   - 点击"AI助手" → "智能文档生成器"

2. **配置生成参数**
   
   **基础设置**：
   - 选择文档类型（学术论文、商业计划书、技术文档等）
   - 详细描述文档需求和内容要点
   - 设置字数要求和详细程度
   
   **高级设置**：
   - 指定写作风格（专业、学术、商务等）
   - 确定目标受众
   - 配置格式要求（是否包含目录、摘要等）
   
   **模板选择**：
   - 选择预设的文档模板
   - 预览模板结构
   - 自定义模板参数

3. **生成流程**
   - 点击"开始生成"启动AI处理
   - 实时查看生成进度和章节预览
   - 等待完整文档生成完成

4. **结果处理**
   - 生成的文档会自动插入编辑器
   - 可以继续编辑和优化内容
   - 保存或导出最终文档

#### 3.3 支持的文档类型

- **学术论文**: 包含摘要、引言、方法、结果、讨论等标准结构
- **研究报告**: 适用于各类研究项目和调研报告
- **商业计划书**: 创业项目和投资申请的完整方案
- **项目方案**: 项目规划和实施方案
- **技术文档**: 软件开发和技术项目文档
- **用户手册**: 产品使用说明和操作指南
- **市场分析**: 行业分析和市场调研报告
- **简历求职信**: 个人简历和求职申请材料

### 4. 🆕 智能排版设计

#### 4.1 功能特色

智能排版设计器能够自动分析文档内容，推荐最适合的排版样式，并提供可视化的设计界面。

#### 4.2 使用流程

1. **打开排版设计器**
   - 确保文档有内容
   - 点击"AI助手" → "智能排版设计"

2. **文档分析**
   
   系统会自动分析：
   - 文档类型和内容特征
   - 文档结构和复杂程度
   - 现有格式化元素
   - 字数和段落统计

3. **样式选择**
   
   **预设样式**：
   - 学术论文：严谨的学术格式
   - 商业文档：现代商务风格
   - 技术文档：清晰的技术格式
   - 现代简约：时尚的设计风格
   
   **自定义设置**：
   - 字体族和字号调整
   - 行间距和段落间距
   - 页边距和对齐方式
   - 标题层级样式
   - 颜色主题选择

4. **AI建议**
   - 启用AI智能分析
   - 获取专业排版建议
   - 查看改进建议

5. **实时预览**
   - 查看排版效果预览
   - 调整参数并实时更新
   - 确认满意后应用到文档

#### 4.3 排版样式说明

**学术论文样式**：
- 字体：Times New Roman, 12pt
- 行间距：1.5倍
- 对齐：两端对齐
- 标题：层级化字号设计
- 适用：学术论文、研究报告

**商业文档样式**：
- 字体：Calibri, 11pt
- 行间距：1.2倍
- 对齐：左对齐
- 色彩：现代商务色调
- 适用：商业计划、项目方案

**技术文档样式**：
- 字体：Consolas等宽字体
- 紧凑的间距设计
- 清晰的层级结构
- 适用：技术文档、API文档

**现代简约样式**：
- 字体：Microsoft YaHei
- 较大的行间距
- 简洁的设计元素
- 适用：现代化文档、演示材料

### 5. 🆕 文档模板库

#### 5.1 模板系统概述

文档模板库提供了丰富的专业模板，用户可以快速创建标准化的文档，也可以自定义和管理个人模板。

#### 5.2 使用模板

1. **打开模板库**
   - 点击"AI助手" → "文档模板库"

2. **浏览模板**
   - 按类别筛选模板
   - 查看模板描述和预览
   - 选择合适的模板

3. **配置参数**
   - 切换到"参数设置"标签
   - 填写模板中的变量值
   - 自定义文档信息

4. **使用模板**
   - 点击"使用此模板"
   - 模板内容会替换当前文档
   - 继续编辑和完善内容

#### 5.3 内置模板介绍

**学术论文模板**：
- 标准的学术论文结构
- 包含：摘要、关键词、引言、方法、结果、讨论、结论、参考文献
- 变量：标题、作者、单位、日期等

**商业计划书模板**：
- 完整的商业计划书框架
- 包含：执行摘要、公司概述、市场分析、产品服务、营销策略、财务预测等
- 适用于创业项目和投资申请

**技术文档模板**：
- 软件项目技术文档标准
- 包含：概述、系统架构、功能说明、API文档、部署指南等
- 适用于开发文档和技术说明

**项目报告模板**：
- 项目管理标准报告格式
- 包含：项目概述、进展状况、风险管理、资源使用等
- 适用于项目汇报和进度跟踪

**会议记录模板**：
- 标准的会议记录格式
- 包含：会议信息、参会人员、议程、决议、行动计划等

**创建新模板**：
1. 点击"新建模板"
2. 输入模板名称和描述
3. 编写模板内容，使用 `{变量名}` 定义可替换变量
4. 保存为自定义模板

**编辑模板**：
- 只能编辑自定义模板
- 修改模板内容和变量定义

**删除模板**：
- 只能删除自定义模板
- 确认删除操作
- 从模板库中移除

## 最佳实践

### 1. 文档创建建议

**选择合适的起点**：
1. **从模板开始**: 对于标准文档类型，使用内置模板可以节省时间
2. **AI生成框架**: 对于复杂文档，先用AI生成器创建基础框架
3. **空白文档**: 对于创意性内容，从空白文档开始

**文档结构规划**：
- 先确定文档的主要章节和逻辑流程
- 使用大纲视图规划文档结构
- 利用AI建议优化章节安排

### 2. AI功能使用技巧

**文本处理策略**：
- 先写出基本内容，再用AI润色优化
- 利用扩写功能丰富关键段落
- 使用总结功能检查内容要点

**提示词优化**：
- 在AI文档生成时，提供详细和具体的需求描述
- 明确指定目标受众和应用场景
- 提供相关的背景信息和约束条件

### 3. 排版设计建议

**样式选择原则**：
- 根据文档用途选择合适的预设样式
- 学术文档优先考虑可读性和专业性
- 商业文档注重现代感和视觉效果

**一致性维护**：
- 使用统一的字体和颜色方案
- 保持标题层级的一致性
- 定期使用AI检查风格一致性

## 常见问题解答

### Q1: AI功能需要网络连接吗？
**A**: 是的，AI功能需要网络连接。编辑器通过API调用云端的AI服务，因此需要稳定的网络连接。

### Q2: 支持离线使用吗？
**A**: 基础的文本编辑功能支持离线使用，但AI辅助功能需要网络连接。离线模式下可以进行文本编辑、格式化和文档管理。

### Q3: 如何配置AI服务？
**A**: 在设置页面中配置API密钥和服务端点。支持OpenAI GPT、百度文心等多种AI服务。

### Q4: 生成的文档质量如何保证？
**A**: 
- AI生成的内容需要人工审核和修改
- 提供详细的需求描述可以提高生成质量
- 建议将AI生成作为起点，后续进行完善

### Q5: 如何处理大型文档？
**A**: 
- 大型文档建议分章节处理
- 利用大纲视图管理文档结构
- 定期保存避免数据丢失

### Q6: 模板可以导出分享吗？
**A**: 自定义模板会保存在本地，可以通过导出功能分享。内置模板无法修改但可以复制创建自定义版本。

### Q7: 支持哪些文件格式？
**A**: 
- 导入：Word(.docx)、Markdown(.md)、HTML(.html)、纯文本(.txt)
- 导出：Word、PDF、Markdown、HTML、纯文本

### Q8: 如何提高AI生成的准确性？
**A**: 
- 提供详细和具体的需求描述
- 明确文档类型和目标受众
- 提供相关的背景信息和示例
- 分步骤生成，逐步完善

### Q9: 版本历史会占用很多空间吗？
**A**: 编辑器有自动清理机制，会定期清理较老的版本。用户也可以手动删除不需要的版本。

### Q10: 如何备份我的文档和设置？
**A**: 
- 文档可以导出为多种格式进行备份
- 自定义模板保存在templates文件夹中
- 设置信息可以通过配置文件备份
