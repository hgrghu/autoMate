# autoMate 项目详细分析报告

## 项目概述

autoMate 是一款基于 AI 的本地自动化工具，采用 "AI+RPA" 的技术架构，能够通过自然语言理解用户需求，智能识别屏幕内容，并自动执行复杂的桌面操作任务。

## 核心功能特性

### 1. 智能任务规划
- **自然语言理解**: 用户只需用中文描述任务需求，无需编程知识
- **AI 驱动规划**: 基于 GPT-4o 等大模型的智能任务分解
- **自主决策**: 根据屏幕内容自动判断下一步操作

### 2. 视觉识别与分析
- **屏幕元素识别**: 基于 OmniParser 和 YOLO 模型识别界面元素
- **视觉标注**: 自动在屏幕截图上标注可操作元素
- **多元素过滤**: 过滤重叠和包含的界面元素，确保识别准确性

### 3. 智能操作执行
- **全界面控制**: 支持任何图形界面的自动化操作
- **精确定位**: 基于坐标和视觉识别的精确点击和输入
- **操作反馈**: 实时显示操作状态和进度

### 4. 安全与隐私保护
- **本地化部署**: 所有操作在本地执行，保护用户隐私
- **可控范围**: 支持屏幕区域选择，限制操作范围
- **即时停止**: 提供全局热键和界面按钮停止功能

## 技术架构

### 核心组件

1. **UI 层** (`ui/`)
   - `main_window.py`: 主界面实现 (PyQt6)
   - `settings_dialog.py`: 设置对话框
   - `tray_icon.py`: 系统托盘图标
   - `theme.py`: 主题管理

2. **自动化控制层** (`auto_control/`)
   - `app.py`: Gradio Web 界面
   - `loop.py`: 主要的采样循环逻辑
   - `agent/`: 智能代理模块
   - `tools/`: 工具集合
   - `executor/`: 执行器

3. **代理系统** (`auto_control/agent/`)
   - `task_plan_agent.py`: 任务规划代理
   - `task_run_agent.py`: 任务执行代理
   - `vision_agent.py`: 视觉识别代理

4. **工具层** (`util/`)
   - `screen_selector.py`: 屏幕区域选择
   - `auto_control.py`: 自动化控制工具
   - `download_weights.py`: 模型权重下载

## 技术实现细节

### 1. 视觉识别流程
```python
# 屏幕截图 → YOLO 检测 → 元素过滤 → 坐标映射 → 视觉标注
screenshot → object_detection → filtering → coordinate_mapping → visual_annotation
```

### 2. 任务执行流程
```python
# 用户输入 → 任务规划 → 屏幕分析 → 操作执行 → 状态反馈
user_input → task_planning → screen_analysis → action_execution → feedback
```

### 3. 支持的操作类型
- 鼠标点击 (左键、右键、双击)
- 键盘输入 (文本、快捷键)
- 滚动操作 (上下滚动)
- 拖拽操作
- 窗口管理

### 4. 模型集成
- **支持的大模型**: GPT-4o, GPT-4o-2024-08-06, o1 等
- **视觉模型**: YOLO (目标检测)
- **OCR 功能**: EasyOCR 文本识别
- **结构化输出**: Pydantic 模型定义
